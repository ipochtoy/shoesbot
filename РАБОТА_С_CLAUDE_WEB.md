# Как работать с Claude Web для больших задач

## Схема работы:

```
1. ТЫ пишешь план → 
2. Я готовлю контекст проекта →
3. CLAUDE WEB делает код →
4. Я проверяю/тестирую/деплою
```

## Шаги:

### 1. Создай план задачи (подробный!)

Напиши что нужно сделать:
- Какие модели добавить
- Какие API использовать
- Какие страницы/формы нужны
- Как должно работать

### 2. Я создам "контекст-пакет" для Claude Web:

Соберу в один файл:
- Структуру проекта
- Ключевые файлы (models.py, views.py и т.д.)
- Существующие паттерны кода
- Инструкции как интегрировать

### 3. Ты даешь Claude Web:

```
[Твой план]

[Контекст-пакет который я создам]

Инструкция: Следуй плану, используй паттерны из контекста. 
Создай новые файлы/измени существующие.
```

### 4. Claude Web создает код

Он выдаст файлы которые нужно добавить/изменить.

### 5. Ты копируешь код на локалку

Вставляешь файлы в проект.

### 6. Я проверяю и деплою:

```bash
./test_bot.sh  # Проверяю что бот не сломан
./deploy_from_local.sh  # Деплою если тесты ОК
```

## Что я подготовлю для Claude Web:

### Файл `CONTEXT_FOR_CLAUDE.md` будет содержать:

1. **Структура проекта:**
   - Какие модели есть (PhotoBatch, Photo и т.д.)
   - Какие views есть
   - Как устроены templates

2. **Паттерны кода:**
   - Как добавлять новые API endpoints
   - Как создавать формы
   - Как работать с моделями

3. **Существующий код:**
   - models.py (модели)
   - urls.py (роуты)
   - Примеры views

4. **Требования:**
   - Не трогать бота (священная корова!)
   - Следовать существующим паттернам
   - Тестировать перед деплоем

## Пример использования:

**Ты пишешь план:**
```
Нужно добавить выгрузку товаров на eBay:
1. Модель для хранения настроек eBay API
2. Форма в админке для заполнения API ключей
3. Кнопка "Выгрузить на eBay" в карточке товара
4. API endpoint который отправляет товар на eBay
...
```

**Я создаю:** `CONTEXT_FOR_CLAUDE.md`

**Ты даешь Claude Web:**
- Твой план
- CONTEXT_FOR_CLAUDE.md
- "Сделай это, не трогая бота"

**Claude Web выдает:** Готовые файлы

**Ты копируешь** на локалку

**Я проверяю/деплою:** Одна команда
